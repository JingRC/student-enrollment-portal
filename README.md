# Student Enrollment Portal

一个基于 HTML、CSS、JavaScript 实现的学生注册页面项目，围绕学生信息登记场景进行设计，兼顾页面美观、交互体验与展示效果。

## 在线体验

GitHub Pages 地址：

https://jingrc.github.io/student-enrollment-portal/

如果刚部署完成时暂时无法访问，通常等待几分钟后刷新即可。

## 项目简介

本项目采用双栏布局设计：

- 左侧区域用于展示页面品牌信息、实时档案卡与校园风采轮播
- 右侧区域用于填写学生注册信息表单

页面不仅实现了基础的学生注册信息录入，还加入了实时预览、密码校验、头像上传、本地草稿保存、填写进度提示、校园图片轮播和信息名片生成功能等前端交互效果，使整个页面更接近完整的静态产品展示页。

## 主要功能

- 支持学生姓名、性别、学号、联系电话、电子邮箱、出生日期、院系、专业、班级、密码和头像等信息录入
- 支持头像本地上传，并在实时档案卡中同步预览
- 根据出生日期自动计算年龄，并显示在实时档案卡中
- 支持密码强度检测与确认密码一致性校验
- 支持填写进度条动态更新和提交按钮状态联动
- 支持本地草稿保存与恢复，减少重复填写
- 支持校园风采图片轮播，并提供左右手动切换按钮
- 支持“生成名片”弹层，根据当前表单信息实时生成信息名片预览
- 支持 6 种预设名片模板切换，包括晨曦校园、专业蓝调、纸感轻奢、森系翠影、绯光映像、夜幕星河
- 支持桌面端保存 PNG 名片图片，支持移动端分享或下载名片图片
- 支持移动端名片弹层适配，预览与模板选择可在手机端正常操作

## 技术实现

### HTML

- 使用语义化结构组织页面，包括主区域、品牌展示区、实时档案卡、轮播模块和注册表单
- 使用多种表单标签实现学生信息录入，例如 `input`、`select`、`fieldset`、`button` 等
- 使用 `dl`、`dt`、`dd` 构建实时档案卡中的信息展示区域

### CSS

- 使用 Grid 和 Flex 实现双栏排版与表单网格布局
- 使用渐变背景、圆角、阴影、模糊和玻璃拟态效果增强页面视觉层次
- 使用动画与过渡效果优化按钮、卡片、输入框和轮播模块的交互表现
- 使用响应式适配保证页面在不同屏幕尺寸下正常显示，并针对移动端优化轮播和名片生成弹层布局

### JavaScript

- 实现表单内容与实时档案卡的同步更新
- 实现年龄自动计算、密码强度提示、确认密码状态提示
- 实现本地草稿保存与恢复
- 实现填写进度动态更新
- 实现校园风采轮播、自动切换与手动切换控制
- 实现基于 Canvas 的信息名片绘制、模板切换和图片导出
- 实现桌面端保存、移动端分享/下载回退等名片导出逻辑
